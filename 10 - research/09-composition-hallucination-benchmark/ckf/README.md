# CKF-compatible experimental exports

This directory contains an experimental CKF-compatible projection of the canonical benchmark cases.

The canonical benchmark format remains `cases/*.json` validated by `schema/case_schema.json`. The files in `ckf/cases/*.ckf.json` are provided to support experiments with compiled knowledge, structured retrieval, GraphRAG-style relation extraction, and agentic workflows.

Status: experimental mapping, not the canonical dataset schema.
Version: 0.4.0-draft.
