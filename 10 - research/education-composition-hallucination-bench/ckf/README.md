# CKF Experimental Representations

This folder contains CKF-compatible experimental JSON packages for each benchmark case.

The benchmark schema remains canonical in `cases/`. The CKF files are intended for experiments with compiled knowledge, agentic retrieval, and relation-aware prompting.

Each CKF file includes the 22 top-level section names listed in the public CKF schema overview: metadata, core_intent, domain_map, entities, concepts, principles, heuristics, decision_rules, procedures, patterns, anti_patterns, causal_chains, contextual_triggers, if_then_rules, exceptions, mental_models, playbooks, qa_pairs, retrieval_chunks, atomic_units, agent_instructions, knowledge_limits, and source_traceability.
