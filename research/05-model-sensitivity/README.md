# Model Sensitivity

**Paper:** *Comparação entre GPT-5 e Gemini 2.5 Pro na Preservação Semântica do CKF Compiler v1.1*

This paper compares how different frontier models affect CKF compilation depth, structure, traceability, fidelity, and semantic preservation.

## Why this matters for CKF

Even when the same source material, compiler version, protocol, and rubric are used, different models may produce CKF outputs with different levels of granularity, abstraction, completeness, and relational depth.

This phase documents model sensitivity as a core factor in CKF compilation and distinguishes research-grade extraction from more compact final-package generation.
